Умови складання іспиту
  • // 1

    Іспит з фізичної підготовленості складається усіма без винятку вступниками на навчання в статусі курсанта.

  • // 2

    Усі фізичні вправи призначені для перевірки виконуються вступником впродовж одного дня.

  • // 3

    Іспит складається двічі під час регіонального відбору та на етапі відбору в Академії.

  • // 4

    Комплексна силова для жінок рахує кількість скручувань за 30 сек плюс віджимання за 30 сек.

Оцінювання іспиту
  • // 1

    Рівень підготовленості визначається сумою балів за виконання трьох вправ.

  • // 2

    Пороговий рівень кожної вправи становить 12 балів, нижче — норматив не зараховується.

  • // 3

    Будь-яка вправа нижче порога анулює результат спроби незалежно від інших показників.

  • // 4

    Іспит складають один раз повторне проходження для підвищення підсумкового балу не дозволяється.
